Chapter 3 Git

本部分主要介绍Git。由于已经存在足够好的Git教程（参见Get started with GitHub | Happy Git and GitHub for the useR，仅作补充说明：

  1. 本教程的大部分Terminal代码，在Rstudio中均有可视化按钮替代（但学习代码依然是重要的）。

  2. Chapter 1-7属于基本认识和设置内容，在电脑上进行设置后不需要重复操作。

  3. Chapter 8 与Rstudio内部的Diff功能相近。

  4. Chapter 9 为日常Git的主要操作流程。

  5. 如无必要，不需要学习SSH部分内容。

  6. 在每次打开合作Project时，首先pull以更新到项目最新版本，并在合作过程中多使用branch，避免override。
